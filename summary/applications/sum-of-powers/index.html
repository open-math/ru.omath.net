Суммы степеней

В математике регулярно приходится иметь дело с самыми разными суммами натуральных чисел в разных степенях:

1+2+3+…+n32+42+52+…+50283+103+123+…+2031 + 2 + 3 + \ldots + n \\ 3^2 + 4^2 + 5^2 + \ldots + 50^2 \\ 8^3 + 10^3 + 12^3 + \ldots + 20^31+2+3+…+n32+42+52+…+50283+103+123+…+203

Эти суммы используются в самых разных разделах математики, особенно в геометрии. Записывать их довольно долго. Чтобы не тратить время и не путаться, ввели обозначение SnkS_n^kSnk​, обозначающее сумму k-ых степеней первых n натуральных чисел:

Сумма степеней чисел

Snk=1k+2k+3k+…+nkS^k_n = 1^k + 2^k + 3^k + \ldots + n^kSnk​=1k+2k+3k+…+nk

Формулы сумм

Сумма чисел

Сумма первых n натуральных чисел:

1+2+3+…+n=n(n+1)21 + 2 + 3 + \ldots + n = \frac{n(n+1)}{2}1+2+3+…+n=2n(n+1)​

В виде многочлена:

Sn1=12n2+12nS_n^1 = \frac{1}{2}n^2 + \frac{1}{2}nSn1​=21​n2+21​n

Числа, получающиеся в результате суммы, называют «треугольными». Формулу легко запомнить: формируем прямоугольник из двух «треугольных» чисел, находим его площадь и делим на 2.

Сумма квадратов

Сумма квадратов первых n натуральных чисел:

12+22+32+…+n2=n(n+1)(2n+1)61^2 + 2^2 + 3^2 + \ldots + n^2 = \frac{n(n+1)(2n+1)}{6}12+22+32+…+n2=6n(n+1)(2n+1)​

В виде многочлена:

Sn2=13n3+12n2+16nS_n^2 = \frac{1}{3}n^3 + \frac{1}{2}n^2 + \frac{1}{6}nSn2​=31​n3+21​n2+61​n

Числа, получающиеся в результате суммы, называют «квадратными пирамидальными». Формулу легко запомнить: формируем параллелепипед из шести «пирамидок», находим его объём и делим на 6.

Доказательство

Смотрите разные способы вывода в статье.

Сумма кубов

Сумма кубов первых n натуральных чисел:

13+23+33+…+n3=n2(n+1)241^3 + 2^3 + 3^3 + \ldots + n^3 = \frac{n^2(n+1)^2}{4}13+23+33+…+n3=4n2(n+1)2​

В виде многочлена:

Sn3=14n4+12n3+14n2S_n^3 = \frac{1}{4}n^4 + \frac{1}{2}n^3 + \frac{1}{4}n^2Sn3​=41​n4+21​n3+41​n2

Формулу легко запомнить: достаточно возвести в квадрат формулу суммы первых n чисел.

Рекуррентная формула

Прямые формулы для расчёта первых n чисел в любой степени существуют, но чем выше степень, тем более трудоёмким становится процесс вывода, потому что для вывода формулы SnkS_n^kSnk​ нужно задействовать все формулы для предыдущих степеней: Snk−1S_n^{k-1}Snk−1​, Snk−2S_n^{k-2}Snk−2​ и так далее…

Рекуррентная формула суммы степеней

Snk=1k+1(nk+1+∑t=2k+1(−1)tCk+1tSnk+1−t)S_n^k = \frac{1}{k+1}\left( n^{k+1} + \sum\limits_{t=2}^{k+1}(-1)^t C_{k+1}^t S_n^{k+1-t} \right)Snk​=k+11​(nk+1+t=2∑k+1​(−1)tCk+1t​Snk+1−t​)
